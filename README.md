# Franz-Plugin

## Instalación
Para instalar este plugin, debes acceder a la carpeta de instalación de Franz. Por defecto suele estar en la siguiente ruta:
> C:\Users\usuario\AppData\Roaming\Franz\Plugins   (Windows 10)
